ADA Assignment 1

Course: Analysis and Design of Algorithms
Language: Python

Tasks Implemented

Task 1: Algorithm Growth Observation

Implemented algorithms to demonstrate different time complexities:

- Constant Time O(1)
- Linear Time O(n)
- Quadratic Time O(n²)
- Logarithmic Time O(log n)

Execution time is measured and visualized using graphs.

Task 2: Best, Average and Worst Case Analysis

Comparison between:

- Linear Search
- Binary Search

Execution time for different input sizes is plotted using matplotlib.

Task 3: Recursion and Recurrence Validation

Implemented recursive algorithms:

- Factorial
- Fibonacci (Naive recursion)
- Fibonacci using Dynamic Programming

Comparison of number of function calls and execution time.

Task 4: Solving Recurrences through Code

Two recurrence relations implemented:

1. T(n) = T(n/2) + n
2. T(n) = 2T(n/2) + n

Number of recursive calls observed to validate theoretical complexity.
